# UNIVERSAL-VEDIO-DOWNLOADER
GitHub par repo banate waqt description mein likhna: *"A simple web-based universal video downloader using Python and Flask."*
# 🚀 Universal Video Saver Pro

Ek simple aur powerful web-based video downloader jo **yt-dlp** aur **Flask** ka upyog karke banaya gaya hai. Is tool se aap lagbhag kisi bhi website se videos download kar sakte hain.

## ✨ Features
- **Wide Support:** YouTube, Instagram, Facebook, aur 1000+ anya sites supported hain.
- **High Quality:** Hamesha sabse acchi available quality download karta hai.
- **Modern UI:** Glassmorphism design aur dark mode ke saath.
- **Fast & Lightweight:** Flask backend ki wajah se bohot tezi se kaam karta hai.

## 🛠️ Installation

Ise apne local machine par chalane ke liye niche diye gaye steps follow karein:

1. **Repository ko Clone karein:**
   ```bash
   git clone https://github.com/krishnashauryayadav-svg/UNIVERSAL-VEDIO-DOWNLOADER
   cd UNIVERSAL-VEDIO-DOWNLOADER
2. **Dependencies Install karein:
    ```bash
    pip install -r requirements.txt
3. **App ko run kare:
    ''' bash
    python app.py
4. **Browser mein kholein:
    Go to http://127.0.0.1:5000
